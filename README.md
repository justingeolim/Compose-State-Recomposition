# Compose-State-Recomposition
CS 501 Midterm #1
Satisfies the following conditions on Jetpack Compose: 

## Create a simple counter screen in Jetpack Compose.
### Requirements:
1.	The screen should:
    o	Display a number starting at 0
    o	Include two buttons:
        	Increment
        	Reset
    o	Update the UI whenever the count changes
    o	The state must survive screen rotation
2.	You must use:
    o	rememberSaveable
    o	mutableStateOf
